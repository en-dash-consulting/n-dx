Three packages, one pipeline

Each package solves a distinct problem. Together, they form a complete AI development workflow.

sourcevision

Analyze

Static analysis engine that maps your codebase. File inventory, import graph, architectural zone detection, and React component catalog.

  • File inventory with roles & metadata
  • Import dependency graph
  • Louvain community detection for zones
  • AI-readable CONTEXT.md output
ndx plan .

rex

Plan

PRD management with hierarchical epics, features, tasks, and subtasks. Scan your project to generate proposals, track progress, and validate dependencies.

  • Hierarchical task tree (epic → subtask)
  • AI-driven proposal generation
  • Dependency tracking & validation
  • MCP server for Claude integration
ndx plan --accept .

hench

Execute

Autonomous agent that picks the next task, builds a brief, and drives Claude in a tool-use loop. Records runs with full transcripts.

  • Autonomous task execution
  • Tool-use loop with Claude API
  • Run history & transcripts
  • Configurable model & turn limits
ndx work .

How it works

A streamlined loop from analysis to execution.

1

Analyze

sourcevision scans your codebase—files, imports, zones, components—and produces a machine-readable context.

2

Plan

rex reads the analysis and generates a structured PRD. Review proposals, accept them, and track progress in a task tree.

3

Execute

hench picks the next task, builds a brief with full context, and ships it—writing code, running tests, committing results.

Get started in seconds

npx n-dx init .

Then run ndx plan . to analyze and ndx work . to start executing.
