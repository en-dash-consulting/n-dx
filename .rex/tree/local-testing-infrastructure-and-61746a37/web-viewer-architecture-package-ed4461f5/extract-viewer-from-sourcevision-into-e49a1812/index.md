## Subtask: Consolidate/merge tasks and features from Rex UI

**ID:** `317a6971-1a95-4ec0-9ee0-24829df8e812`
**Status:** completed
**Priority:** low

Add the ability to consolidate (merge/collapse) items from within the Rex web UI. A user should be able to select multiple tasks under a feature and merge them into one, or select multiple features under an epic and consolidate them. Use cases: cleaning up redundant tasks generated by `rex analyze`, combining tasks that turned out to be the same work, rolling up granular subtasks into a single task after the fact. **Acceptance Criteria**

- Can select multiple features within an epic and consolidate
- Preview shown before merge commits
- Merged item preserves acceptance criteria and descriptions from all sources
- API endpoint or MCP tool supports the consolidation operation
- Execution log records the merge with original item IDs
