Item NDX-001: Windows Install Reliability

Type: Bug / Story
Priority: Suggested P0
Status: Draft

Description:
Improve the Windows installation and setup experience so users can reliably install and begin using N-DX.

User Story:
As a Windows user, I want a reliable installation experience, so that I can successfully start using N-DX without troubleshooting setup failures.

Acceptance Criteria:

Given a Windows user follows the recommended install path, when installation completes, then N-DX is available and runnable.
Given installation prerequisites are missing, when setup is attempted, then the user receives clear guidance.
Given installation fails, when an error is shown, then it provides enough information to recover or report the issue.

Notes:
The supported path should be whichever approach is most reliable and effective after validation. Windows reliability is important because many target enterprise users are expected to be on Windows machines.
